
# Main used Technique
- Video with controls and multiple vdieo sources ```<video controls> <video>```, ```<source src="" type="">```
- List
```
<ul>
  <li>
			<ul>
				<li> </li>
				<li> </li>
			</ul>
  </li>
  <li></li>
</ul>
```
- Table
```
<table>
  <tr>  <th></th>  <th></th>  </tr>
  <tr>  <td></td>  <td></td>  </tr>
</table>
```
- Form and label:
```
<form>
  <label></label> <input>
  <label></label> <input>
</form>
```
- Fieldset:
```
<fieldset>
  <legend>title</legend>
  <label></label> <input>
</fieldset>
```
